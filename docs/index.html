01 · COS'È

Costruita con Electron — finestra propria, icona nel taskbar, avvio rapido. Nessun browser da aprire a mano.
100% offline
Il server Next.js gira embedded nell'app. I tuoi file non lasciano mai il PC — eccetto le funzioni AI opzionali.

02 · MODULI
M-01
LocalSync Bridge Socket.io · Express · PWA

Trasferisci file, testo e credenziali tra PC e smartphone sulla rete di casa — senza cavi, senza cloud. Scansiona il QR code e sei connesso in secondi.

  • Stream — appunti condivisi in tempo reale
  • Cloud Transfer — file da smartphone a PC
  • Accesso via QR code, nessun URL da digitare
  • HTTPS locale, PIN di accesso opzionale
  • PWA installabile su iPhone e Android
  • Multi-utente con ruoli admin / user
M-02
FileForge Studio Next.js · TypeScript · 20 tools

Suite locale con 20 strumenti per PDF, immagini e utilità. Comprimi, firma, converti, genera QR — tutto offline, nessun upload a terzi.

  • PDF: comprimi, unisci, dividi, organizza pagine
  • PDF ↔ Word · PDF ↔ Immagini · Firma + vault
  • Immagini: converti JPG/PNG/WEBP/HEIC, rimuovi sfondo AI
  • Genera QR code, barcode, password sicure
  • Pulizia metadati, analizzatore PDF
  • AI → SVG, testo → immagine tipografica
