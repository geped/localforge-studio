# **App Name**: FileForge

## Core Features:

- PDF Utility Suite: Upload a PDF file and access various tools including compression, merging with other PDFs, splitting into separate documents or page ranges, reordering pages with drag-and-drop, and flipping individual pages.
- PDF and Word Conversion: Convert PDF documents to editable Word files (.docx) and convert Word files to PDF documents. Support for common styling and formatting to be preserved during conversion.
- Image Format Converter: Upload an image and convert it between popular formats: JPEG, PNG, and WEBP. The tool allows selecting the desired output format before downloading the converted image.
- AI Background Removal Tool: Utilize the remove.bg API to intelligently remove backgrounds from uploaded images, providing a clean subject for various uses. Users upload an image, and the tool processes and offers a downloadable image with a transparent background.

## Style Guidelines:

- Primary Color: #2966A3 (a modern, moderate blue, suggesting precision and reliability for a digital tool suite). This color provides a focal point and evokes trust and efficiency.
- Background Color: #ECF0F2 (a very light, subtle blue-grey, maintaining a clean and minimalist canvas, preventing visual clutter for complex tools).
- Accent Color: #26BFBF (a vibrant turquoise that stands out to highlight interactive elements, calls to action, and important notifications, drawing the user's attention effectively).
- Body and headline font: 'Inter' (a versatile grotesque-style sans-serif for exceptional readability across all content, supporting a clean, modern, and objective feel for the application).
- Use clear, concise, and modern SVG icons for all tools and actions, ensuring scalability and consistent visual representation. Icons should be intuitively recognizable, avoiding ambiguity.
- Implement a clean, responsive, and intuitive layout that adapts seamlessly to different screen sizes. Utilize ample whitespace to create visual separation and improve user focus, promoting ease of use.
- Incorporate subtle and functional animations for user feedback, such as loading indicators for file processing or subtle transitions for UI state changes. Animations should be swift and not detract from productivity.
